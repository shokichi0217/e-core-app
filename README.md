# e-core system | 自己発注システム

有限会社馬場燃料店（長崎市）のスタッフおよび提携企業向けの、Googleスプレッドシート連携型・自己発注Webアプリケーションです。

---

## 🚀 特長

1. **簡単ログイン**: 管理者と提携担当者を選択するだけのシンプルログイン。
2. **リアルタイム同期**: 注文登録、ステータス更新、お気に入り商品、商品マスタがGoogleスプレッドシートと双方向で自動リアルタイム同期。
3. **お気に入り商品機能**: よく注文する商品を担当者ごとに登録し、ワンクリックで注文明細に追加可能。
4. **スマート検索 & 単価自動計算**: 商品マスタから型番やメーカーでオートコンプリート検索し、メーカー標準単価や特別価格を自動計算。
5. **Excelインポート**: 管理者画面から商品マスタのデータをExcel形式で一括インポート可能。
6. **レスポンシブデザイン**: PCだけでなく、タブレットやスマートフォン（現場）からも快適に操作可能。

---

## ⚙️ 導入・設定手順（管理者向け）

本アプリは静的HTML（シングルページアプリケーション）として動作するため、サーバーの維持費は一切かかりません。

### 1. Googleスプレッドシートの準備
1. スプレッドシート（https://docs.google.com/spreadsheets/d/1-AHVrFgQSZTYQuavRWqxJhBRKCNqEFoDD45rb9PU_Ew/edit）を用意します。
2. アプリの「⚙️ 連携設定」にデプロイしたGAS URLを入力すると、シート内に `Orders`, `Products`, `Favorites` の3つのシートが自動作成されます。

### 2. Google Apps Scriptのデプロイ
1. スプレッドシートのメニューから「拡張機能」＞「Apps Script」を開きます。
2. エディタ内の既存のコードをすべて削除し、プロジェクト内の `Code.gs` の内容を貼り付けて保存（Ctrl + S）します。
3. 右上の「デプロイ」＞「新しいデプロイ」をクリックします。
4. 「ウェブアプリ」を選択し、以下のように設定します：
   - 次のユーザーとして実行: **「自分」**
   - アクセスできるユーザー: **「全員」**（CORS連携のために必須）
5. デプロイを実行し、発行された **「ウェブアプリのURL」** (末尾が `/exec` のもの) をコピーします。
6. アプリ起動後、画面左下の「⚙ 連携設定」に入力して保存します（デフォルトで事前設定済みです）。
